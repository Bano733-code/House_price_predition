# 🏠 California Housing Price Prediction


## Project Overview

This project uses Machine Learning to predict California housing prices based on different features such as location, number of rooms, population, and income.

A Random Forest Regression model is trained on the California Housing Dataset and deployed using Streamlit.


## Features

- Data preprocessing
- Feature scaling
- Machine Learning regression model
- House price prediction
- Interactive Streamlit web application


## Dataset

Dataset:
California Housing Dataset

Features include:

- Longitude
- Latitude
- Housing Median Age
- Total Rooms
- Total Bedrooms
- Population
- Households
- Median Income


Target:

- Median House Value


## Machine Learning Model

Algorithm Used:

Random Forest Regressor


## Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Data Scaling
6. Model Training
7. Model Evaluation
8. Deployment using Streamlit


## Evaluation Metrics

The model was evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score


## Installation


Clone repository:

```
git clone https://github.com/Bano733-code/House_price_predition
```


Install dependencies:

```
pip install -r requirements.txt
```


## Run Application


Run Streamlit app:

```
streamlit run app.py
```

The application takes house information as input and predicts the estimated house price.
